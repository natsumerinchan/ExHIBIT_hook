# ExHIBIT引擎 hook补丁

## 实现功能
- 1、优先从rld_chs读取脚本
- 2、字体修改为黑体
- 3、日志系统
- 4、标题修改
- 5、编码修改为GBK

## 使用方法
理论上适用于所有未加壳的ExHIBIT引擎游戏，安装<br>
Visual Studio 2022后运行build.bat编译，然后<br>
使用CFF Explorer导入dll。
